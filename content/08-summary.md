Summarize the three learnings from the presentation:

1. **The System Prompt** — Claude Code follows hidden instructions that shape its behavior. You can read and influence them.
2. **The Context Window** — When your context gets full, the model gets stupid. Start fresh when things go sideways.
3. **The Agentic Loop** — Think in loops. Give it the right tools.

Tie back to the core message: "Any technology you understand is no longer magic."
