# How Does Claude Code Work?

**A 10-minute talk demystifying Claude Code for everyday users.**

Presented by Dominik Grusemann (Co-Founder, [Marbles AI](https://usemarbles.ai)) at the [Claude Code for Everyone](https://luma.com/5abx5e5u?tk=yEGUVl) Munich Meetup.

## Core Message

> "Any technology you understand is no longer magic."

Most people use Claude Code daily but treat it as a black box. This talk opens it up — three concepts, ten minutes, one story.

## Audience

UX designers, product managers, consultants, vibe coders, copywriters, and decision-makers — all already using Claude Code, none necessarily technical.
