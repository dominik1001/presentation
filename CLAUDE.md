# Claude Code for Everyone, Munich

This repository includes my presentation for the meetup "Claude Code for Everyone".

## The Talk

**Title:** How does Claude Code work? …and what can we learn from it.
**Speaker:** Dominik Grusemann, Co-Founder Marbles AI
**Event:** Claude Code for Everyone — Munich Meetup
**Venue:** Accenture Song, Balanstraße 73, 81541 Munich
**Slot:** 19:20, Monday — 10 min talk + 5 min Q&A

## Audience

UX designers, product managers, consultants, vibe coders, copywriters, decision-makers.
All already use Claude Code. None are necessarily technical.
Goal: reduce fear, build understanding. Not a tutorial.

How often do you use Claude Code?
- Daily: 44%
- Regular: 24%
- New but highly interested: 17%
- Occasional user: 14%

## Core Message

> "Any technology you understand is no longer magic."

The audience uses Claude Code daily but sees it as a black box.
The talk demystifies it. Three concepts, ten minutes, one story.
